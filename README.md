# Smart Image Compressor Pro

Smart Image Compressor Pro is a modern offline desktop application built using Python that allows users to resize, compress and convert images with an easy to use graphical interface.

The application focuses on maintaining high visual quality while reducing file size, giving users full control over dimensions, target size, and output format.

# Features

* 🖱️ Fully interactive GUI with mouse and keyboard support
* 📂 Drag & drop image upload
* 🖼️ Before and after image preview
* 📏 Custom width and height resizing
* 📦 Smart image compression with quality optimization
* 📊 Compression percentage and size reduction details
* 📁 Batch image compression support
* 🌗 Dark and light theme modes with saved preferences
* 🔄 Convert images between JPG, JPEG, PNG, WEBP and PDF
* 💾 Safe offline processing without uploading files
* 🛡️ Original files remain untouched

# 🛠️ Built With

- Python
- CustomTkinter
- Pillow (PIL)
- tkinterdnd2

## 🎯 Goal

This project was created to build a practical desktop utility while improving skills in GUI development, image processing, file handling and application design.
